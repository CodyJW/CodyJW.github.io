Artifact

The artifact selected for this enhancement is my CS360 weight tracking Android application. The primary files enhanced were WeightGridActivity.java and WeightTrackerDBHelper.java.

Overview

The original application allowed users to create an account, log in, track daily weight entries, set a goal weight, and receive optional SMS notifications. I focused on improving the software design and maintainability of the daily weight tracking screen.

Enhancements Made

  • Moved repeated validation logic into reusable helper methods.
  • Centralized grid refresh behavior into one method.
  • Added a delete confirmation dialog to reduce accidental data loss.
  • Improved date validation using MM/DD/YYYY format.
  • Updated date sorting in the database helper.
  • Replaced hardcoded values with named constants.

Course Outcomes

This enhancement supports Course Outcomes 2, 3, and 4 by improving user interaction, strengthening the design of an existing computing solution, and applying Android Studio, Java, and SQLite to improve a working application.
