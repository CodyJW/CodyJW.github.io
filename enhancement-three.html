Artifact

The artifact selected for this enhancement is CRUD_Python_Module.py, a Python module originally created in CS340: Client/Server Development.

Overview

This module provides database access for an animal shelter dashboard application. It connects to MongoDB using PyMongo and supports create, read, update, and delete operations through the AnimalShelter class.

Enhancements Made

  • Removed leftover test code that did not support the main module.
  • Added database connection verification using a MongoDB ping command.
  • Added an is_connected() helper method to prevent operations without an active connection.
  • Updated CRUD methods to fail safely when the database is unavailable.
  • Replaced broad exception handling with database-specific PyMongoError handling.
  • Improved validation before create, read, update, and delete operations.
  • Added optional projection support to the read() method.

Course Outcomes

This enhancement supports Course Outcome 4 by improving a database access layer using Python, MongoDB, and PyMongo. It also supports Course Outcome 3 because the enhancement required evaluating the original module and improving how it handles connection failures, invalid input, and database errors. Course Outcome 5 is also supported through safer validation, connection checks, and protection against unintended delete operations.
