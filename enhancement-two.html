Artifact

The artifact selected for this enhancement is LogAnalyzer.java, a standalone Java program created to practice file input and basic data storage.

Overview

The original version of the program read log entries from a text file, stored each line in a list, and printed the raw log data to the console. For this enhancement, I expanded the program into a log analysis tool that processes structured log data and identifies failed login patterns.

Enhancements Made

  • Added parsing logic to separate IP addresses and event types.
  • Filtered log entries to focus on failed login attempts.
  • Used a HashMap to count failed attempts by IP address.
  • Converted HashMap entries into a sortable list.
  • Sorted results from highest to lowest failed attempt count.
  • Flagged IP addresses that met or exceeded the suspicious threshold.
  • Printed a summary report with total failed attempts.

Course Outcomes

This enhancement supports Course Outcome 3 by applying algorithmic thinking and data structure choices to solve a log analysis problem. It also supports Course Outcome 4 because the enhanced program produces a more useful summary report instead of only displaying raw file contents. There is also a light connection to Course Outcome 5 because the program identifies suspicious login patterns.
